기출 분석 및 교재 선정
2024~2025 미사강변중 영어 기출 시험지를 분석하여 출제 경향을 파악했습니다. 동아(윤정미) 교과서의 출제 포인트를 학년별로 정리하고, 전국 동일 교과서 사용 학교의 기출 패턴까지 교차 검증하여 올해 출제 가능성이 높은 유형을 특정했습니다.
4월 첫째 주
본문 10단계 워크북으로 지문 완전 습득
1과, 2과 본문을 10단계로 분해한 워크북 통합본을 활용했습니다. 영문 해석 → 빈칸 채우기(우리말) → 빈칸 채우기(영어) → 배열 → 전문 쓰기까지, 본문의 모든 문장을 학생 스스로 재구성할 수 있을 때까지 반복 훈련했습니다.
4월 둘째 주
자체 문법 워크북 집중 훈련
1과 문법: to부정사 형용사적 용법(명사 뒤 수식, 전치사 동반 규칙)과 make+목적어+형용사(5형식)를 개념 정리 → 기초 연습 → 실전 적용 3단계로 연습했습니다. 2과 문법: 간접의문문(의문사+주어+동사 어순)과 want+목적어+to부정사를 동일한 3단계 구조로 훈련했습니다.
4월 셋째 주
빈출 실전 문제 25제 풀이 + 직전 요약노트 정리
1과, 2과 각각 리마크 자체 제작 실전 대비 문제 25제를 풀고, 오답을 분석했습니다. 동시에 직전 요약노트로 Communication 표현, Grammar 핵심, Vocabulary 영영풀이를 최종 정리했습니다. 자체 단어장으로 1~2과 전체 어휘 60여 개를 3회 이상 테스트했습니다.
4월 23일 (시험 5일 전)
리마크 자체 제작 동형 모의고사 실전 시뮬레이션
실제 시험과 동일한 형식(선택형 20문항 + 논술형 4문항, 100점 만점)의 동형 모의고사를 자체 출제하여 실전 환경에서 풀었습니다. 출제 유형, 배점, 난이도를 실제 시험에 최대한 맞추어 설계했으며, 이 모의고사의 서답형 정답 4개가 실제 시험에서 그대로 출제되었습니다.

왜 서답형이 중요한가?

서답형(논술형)은 학생이 직접 영어 문장을 써야 합니다. 객관식처럼 찍을 수 없고, 정확한 어순과 어형 변화를 알아야만 정답을 쓸 수 있습니다. 리마크는 동형 모의고사에서 실제 시험과 동일한 문법 포인트, 동일한 정답 형태를 미리 연습시켰기 때문에, 학생들이 시험장에서 당황하지 않고 자신 있게 답을 작성할 수 있었습니다.

출제 문법 유형 — 전 유형 커버

실제 시험에 출제된 4대 핵심 문법이 리마크 대비 자료에서 모두 집중 훈련되었습니다.

✓ to부정사 형용사적 용법

명사 뒤에서 수식하는 to부정사

전치사 동반 규칙 (a chair to sit on)

실제 시험 출제: place to park
리마크 연습: 자체 문법 워크북 1과 + 동형 모의고사

✓ make + O + C (5형식)

목적어를 ~하게 만들다

목적격 보어에 형용사 (부사 X)

실제 시험 출제: made us nervous
리마크 연습: 동형 모의고사 + 자체 문법 워크북 1과

✓ 간접의문문

의문사 + 주어 + 동사 어순

think/believe와 함께 쓸 때 의문사 전치

실제 시험 출제: why people like Snow White / where she lives
리마크 연습: 자체 문법 워크북 2과 + 동형 모의고사

✓ want + O + to-V

목적어가 ~하기를 원하다

hope 구문과의 전환

실제 시험 출제: 서답형 + 선택형 다수
리마크 연습: 동형 모의고사 (hope → want 전환 훈련)
