동형 모의고사와 동일 패턴

왜 은가람중 서답형이 까다로운가?

은가람중 논술형 1번은 단일 문법이 아닌 to부정사 + 사역동사 + so that을 하나의 문장에 결합하는 통합형입니다. 개별 문법만 연습한 학생은 13~15단어 조건에 맞춰 세 구문을 자연스럽게 합치기 어렵습니다.

리마크는 Grammar Build Up에서 개별 문법을 완벽히 익힌 후, 동형 모의고사에서 복합 조건 영작을 연습하는 단계적 접근으로 대비했습니다. 덕분에 학생들은 조건을 하나씩 충족하며 문장을 완성할 수 있었습니다.

출제 문법 유형 — 전 유형 커버

✓ to부정사 형용사적 용법

→ Grammar Build Up 1과에서 반복 영작

✓ 사역동사 (make/let/have)

make + O + V원형 / 형용사

논술형 1: make him understand
선택형: had us finishing(X) 오류 판별
→ Grammar Build Up 1과에서 패턴별 훈련

✓ 간접의문문 (whether / 의문사)

whether + S + V / what kind of + S + V

논술형 2: whether they have / what kind of music
선택형: whether I really enjoyed it
→ Grammar Build Up 2과 + 동형 모의고사

✓ so that + 관계대명사

"~할 수 있도록" / that/which/who 구분

논술형 1: so that I can make him understand
선택형: food that makes you feel good
→ Grammar Build Up 2과 + 내신기출
